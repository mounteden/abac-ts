| Version | Supported          |
| ------- | ------------------ |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

**Do not open a public issue.**

Instead, please use [GitHub's private vulnerability reporting](https://github.com/mounteden/abac-ts/security/advisories/new) to submit your report. You will receive an acknowledgement within 48 hours and a detailed response within 7 days.

Please include:

- A description of the vulnerability
- Steps to reproduce the issue
- The potential impact
- Any suggested fix (optional)
